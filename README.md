# Fox Market

Fox Market is a digital ecommerce market-place for users to buy and sell high-quality digital goods.

Built with:
- Next.js
- TypeScript
- tRPC
- TailwindCSS
- PayloadCMS
- MongoDB


Inspired from [@joschan21](https://www.github.com/joschan21)'s Hungry Hippo project
